# Chá de Bebê & Casamento — Anna, Jacopo & Daniel

Site estático para o chá de bebê e casamento de Anna & Jacopo, celebrando a chegada do Daniel.

## Como rodar localmente

Abra `index.html` diretamente no navegador, ou use qualquer servidor HTTP estático:

```bash
# Python 3
python3 -m http.server 8080
# depois acesse http://localhost:8080
```
